Public Page

This is a public page that anyone can access without authentication.

In this demo, we're using MagicLink to protect certain resources while keeping others public.

Demo Navigation:
  • Public Area (Current) - No authentication required
  • Protected Area - Requires MagicLink authentication
  • MagicLink Admin Service - Generate tokens (separate service)
How it works: When you try to access the protected area, you'll be shown an access denied message with a link to the admin service. This is a completely separate service running on its own port (8080). Generate a token for /protected/* and then use the magic link to access the protected area.
Security Best Practice: The admin service is intentionally separate from the application. In a production environment, the admin service would be on a private network, accessible only to authorized personnel.
